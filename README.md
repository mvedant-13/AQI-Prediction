# AQI Prediction

## Problem Statement
A machine learning model to predict Air Quality Index (AQI) 
from pollutant and weather sensor readings across Delhi-NCR.

## Dataset
- 6 years of hourly data (2020–2025)
- 23 stations across 5 cities
- 25 features including pollutants and weather variables

## Tech Stack
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn
